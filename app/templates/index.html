This app uses /iss/engines/[engine]/markets/[market]/securities/[security]/candles endpoint of MOEX Informational Statistical Server to get OHLCV (open, high, low, close, volume) data for different securities. You need to know engine, market and security params to use this app.

All possible engine and market MOEX ISS names - /iss/index.

You can find engine and market parameters for your security using this request:
/iss/securities?q=[security]
E.g., the value of the column 'group' is 'stock_shares' for q=GAZP. Use stock as engine shares as market. Other parameters of this end point - /iss/reference/5.

After registration you can add / update / delete / reorder settings to fetch your favourite securities, look at its candle charts with tunable sizes and get info about best possible historical long and short deals. There's one limitation in this version of the app - you can fetch a maximum of only 500 last time intervals for a single security. But it's enough to catch current trend.

You can find more information about MOEX Informational Statistical Server and its RESTful API using this link.
